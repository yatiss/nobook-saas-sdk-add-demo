# NOBOOK SDK 实验加试 demo

appid 与 appSecret 需跟nobook申请,然后在 server.js 里配置

#### sdk及demo项目位置
* [实验加试SDK](https://github.com/yatiss/nobook-saas-sdk)

## demo运行方式
```bash
$ yarn
$ yarn start
```
或
```bash
$ npm i
$ npm start
```
访问地址 http://localhost:5050/
